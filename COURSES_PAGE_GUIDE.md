## 📍 How to Navigate

### Step 1: Go to Courses
In the dashboard sidebar, click **"Courses"**

### Step 2: Choose Your Tab
- **"My Enrollments"** - See courses you're registered for
- **"Course Catalog"** - Browse all 605 available courses

### Step 3: Search or Filter (Catalog Tab)
- **Search box**: Type course code (e.g., "CS-201") or name
- **Department filter**: Select a department from dropdown
- Results update instantly

### Step 4: View Course Details
Click any course card to see:
- Course code
- Full course name
- Credits
- Department
- Course level
- Description

## 💡 Tips & Tricks

### Finding Courses
- **By Code**: Type "AR-501" or just "AR" to find Robotics courses
- **By Name**: Search for "Robotics" or "Algorithm"
- **By Department**: Use the dropdown to filter

### Understanding the Tabs
| Tab | What You See | Can Do |
|-----|-------------|--------|
| My Enrollments | Your registered courses | View grade, semester, status |
| Course Catalog | All 605 available courses | Search, filter, explore |

### Stats Cards (Top of page)
- **Credits Completed**: Total earned credits from passed courses
- **Credits In Progress**: Current semester courses
- **Available Courses**: Total courses in database

## 🆘 Troubleshooting

### Courses not showing?
1. Make sure you're on the "Course Catalog" tab
2. Try clearing the search box
3. Reset the department filter to "All"

### Search not working?
- Search is case-insensitive (e.g., "cs-201" = "CS-201")
- Searches both course code AND name
- Try shorter search terms

### Modal not opening?
- Click directly on the course card
- Make sure browser allows popups
