# 🎓 Quick Start Guide - Instructor Integration

## Get Started in 2 Minutes ⚡

### What's New?
The DegreePlanner now shows instructor information for all courses. This helps you:
- ✅ Know who teaches before enrolling
- ✅ See instructor contact info (email, phone, office)
- ✅ Identify primary vs secondary instructors
- ✅ Make informed course selection decisions

---

## Where to Find Instructors

### 1. **Courses Page** 🎯 Main View
```
Navigate: Dashboard → Courses → My Courses Tab
├─ Expand any semester
├─ Look for instructor name next to course title
└─ Click course for full instructor details
```

**What you see:**
- Instructor name (compact view)
- Full details in modal (designation, email, phone, office)
- Primary instructor marked with badge

### 2. **Import Courses Page** 📋 Planning View
```
Navigate: Dashboard → Import Courses
├─ Select your branch and semester
├─ See instructor for each available course
└─ Use to plan your course schedule
```

**What you see:**
- Primary instructor for each course
- Helps identify good instructors before selecting
- Makes course planning easier

### 3. **Timetable Page** 📅 Schedule View
```
Navigate: Dashboard → Timetable
├─ Add instructor name when creating schedule
├─ Keep track of who teaches what
└─ Organize your class schedule
```

**What you see:**
- Field to enter instructor name
- Helps track who you have for each class
- Integrates with your personal schedule

## How to Use

### View Instructor Details
```
1. Go to Dashboard → Courses
2. Expand the semester you want
3. Find your course in the list
4. Look for instructor name below course title
5. Click the course card to see full details
```

### See All Instructors for a Course
```
1. Click on the course you're interested in
2. A modal will open with course details
3. Scroll to "Instructors" section
4. See all instructors teaching this course
```

### Contact an Instructor
```
1. Find the instructor in the list
2. Click on the email icon → Opens email client
3. Click on the phone icon → Shows phone number
4. Click on the office → Shows office location
```

### Plan Before Importing
```
1. Go to Dashboard → Import Courses
2. Select your branch and semester
3. Browse available courses
4. See instructor for each course
5. Make informed selection decisions
```

## Frequently Asked Questions

### Q: Where do I find instructor info?
**A:** Go to Courses page → Expand semester → See instructor name next to course title → Click course to see full details

### Q: Can I contact the instructor?
**A:** Yes! Click the email icon to send an email or phone icon to see their phone number

### Q: How do I know the primary instructor?
**A:** Look for the "Primary" badge next to the instructor name in the detailed view

### Q: Why are there multiple instructors for some courses?
**A:** Some courses have multiple instructors. The primary instructor (marked with badge) is the main one

### Q: How often is instructor data updated?
**A:** Data is from the current semester. Updates are made annually at the start of new academic year

### Q: Can I save instructor info?
**A:** Copy email/phone to your contacts or save office location in maps app

### Q: What if an instructor's info is wrong?
**A:** Contact the department admin to update the database

### Q: Why don't I see instructors for some courses?
**A:** Those courses might not have instructor assignments yet. Check with department

## Troubleshooting

### Problem: No instructors showing
```
Solution:
  1. Refresh the page (Ctrl+R)
  2. Wait a moment for data to load
  3. Check if course code is correct
  4. Try a different course
  5. Clear browser cache if still not working
```

### Problem: Slow loading
```
Solution:
  1. First load is normal (~600ms)
  2. Subsequent loads are instant (~50ms)
  3. Check internet connection
  4. Try clearing browser cache
  5. Try refreshing the page
```

### Problem: Can't click email/phone
```
Solution:
  1. Make sure you're not on mobile browser
  2. Check if link is actually clickable (blue underline)
  3. Try again in a different browser
  4. Copy and paste manually if needed
```

### Problem: Wrong instructor showing
```
Solution:
  1. Refresh the page
  2. Check course code is correct
  3. Contact admin if issue persists
  4. Check if data was recently updated
```
